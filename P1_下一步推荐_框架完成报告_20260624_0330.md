### 4. 英文→中文翻译（规则词典）

**实现**: 金融术语词典映射（50+术语）

**关键方法**:
```python
# 规则翻译（关键词替换）
title_cn = rule_translate(title_en)
```

**翻译词典** (`FINANCIAL_TERMS_DICT`):
- 市场术语: 'bullish'→'看涨', 'bearish'→'看跌', 'surge'→'飙升', ...
- 政策术语: 'Fed'→'美联储', 'rate cut'→'降息', 'PBOC'→'中国人民银行', ...
- 行业术语: 'semiconductor'→'半导体', 'GPU'→'图形处理器', 'EV'→'电动汽车', ...
- 财报术语: 'earnings'→'财报', 'beat'→'超预期', 'guidance'→'指引', ...
- 大宗商品: 'crude oil'→'原油', 'copper'→'铜', 'gold'→'黄金', ...
- 地缘政治: 'Hormuz'→'霍尔木兹', 'sanctions'→'制裁', 'tariff'→'关税', ...

**示例**:
```
原文: "Fed rate cut expected, bullish for semiconductor stocks. AI and GPU demand surge."
译文: "美联储 降息 预期，看涨 半导体 股票。人工智能 和 图形处理器 需求 飙升。"
```

**限制**: 仅为关键词替换，不完整翻译。LLM翻译待集成。

核心决策逻辑 — InOut字段(主买净额)

InOut > 0 (主买为正) → GENUINE_INFLOW → HOLD (真实买入,可能续涨/连板)

InOut < 0 + 涨幅 > 3% → WASH_TRADE → SELL (对倒出货,左手倒右手)

涨停封住 → HOLD (等连板,不轻易出清,避免蜀道装备式错误)

跌破止损线 → 清仓 (纪律止损)
600118中国卫星 7/10案例:
涨停+10% BUT InOut=-12亿(主买净额为负) → V54对倒检测: 主力+21.9%但主买-22.6% → 背离44.5% → SELL信号
这正是系统要捕捉的"利好出货"模式 — 涨停是假象,主力在出货

蜀道装备教训修正

错误模式(蜀道装备): 单一信号 → 全仓出清 → 错失后续连涨收益

修正模式(V55): 多时间点监控 → 分批操作 → 给股票自证机会
- 09:45: InOut<0 → 先卖50%(不全仓)
- 10:30: V54=GENUINE → 持有剩余50% / V54=WASH → 清仓剩余
- 14:55: 最终判决 → HOLD过夜或清仓

核心原则: 宁可少赚,不可错杀; 宁可少亏,不可死扛

## 🔧 核心技术亮点

### 1. 实时同步监听
- **技术方案**: `wxauto`库（基于UIAutomation）
- **监听频率**: 每5秒检查一次文件传输助手消息
- **消息延迟**: ≤5秒
- **支持消息类型**: 文字/图片/文件/链接

### 2. 崩溃自动重启
- **重启机制**: 监听器中集成重启逻辑（`main()`函数中的`while True`循环）
- **最大重启次数**: 10次/24小时（自动重置）
- **重启延迟**: 5秒
- **健康检查**: 每5分钟自检一次（数据库连接/消息处理）

### 3. 后台运行（无窗口）
- **启动方式**: `pythonw.exe`（无窗口）
- **启动脚本**: `start_wechat_sync.bat`
- **停止脚本**: `stop_wechat_sync.bat`
- **进程名**: `pythonw.exe`

### 4. 系统启动自动运行
- **方案**: Windows任务计划程序（`schtasks`命令）
- **触发器**: 系统启动时（延迟10秒）
- **权限**: SYSTEM账户（无需用户登录）
- **配置指南**: `wechat_task_scheduler_guide.md`

### 5. 自动化任务监控
- **任务ID**: `automation-1782242953720`
- **频率**: 每小时检查一次
- **功能**: 检查监听器是否运行 → 如果未运行则启动
- **报告**: 生成健康检查报告

## 📋 安装步骤

### 1. 安装wxauto库
```bash
pip install wxauto -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2. 配置Windows任务计划（系统启动时运行）
以**管理员身份**运行以下命令：
```batch
schtasks /create /tn "V13.2_微信实时同步" /tr "E:\WorkBuddy_dot_workbuddy\Claw\start_wechat_sync.bat" /sc onstart /ru SYSTEM /f
```

### 3. 手动启动监听器（测试）
```batch
cd /d E:\WorkBuddy_dot_workbuddy\Claw
start_wechat_sync.bat
```

### 4. 检查运行状态
